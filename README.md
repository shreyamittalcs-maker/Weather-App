Weather App 🌦️

A simple and responsive Weather App built using HTML, CSS, JavaScript, jQuery, and OpenWeatherMap API.
This app allows users to search for any city and get real-time weather information including temperature, weather condition, wind speed, and current date & time.

🚀 Features
🌍 Search weather by city name
🌡️ Real-time temperature display
☁️ Weather condition description
💨 Wind speed information
📅 Current date and time using Moment.js
🎨 Beautiful and responsive UI
⚡ Fast weather data fetching using OpenWeatherMap API

🔑 API Setup

This project uses the OpenWeatherMap API.

Steps to get your API key:
Visit the official website:
OpenWeatherMap
Create an account
Generate your free API key
Replace the API key inside script2.js
